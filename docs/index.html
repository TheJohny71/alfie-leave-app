Getting Started

The Alfie Design System is built to create a consistent, accessible, and modern leave management application. It follows Apple's design principles while maintaining enterprise-grade functionality.

Installation

git clone https://github.com/TheJohny71/alfie-leave-app.git
cd alfie-leave-app

Usage

Each component is built using React and Tailwind CSS, ensuring maximum flexibility and maintainability. Check individual component documentation for specific implementation details.
