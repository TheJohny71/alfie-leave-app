Overview

Usage Guidelines

  • Use primary buttons for main actions
  • Secondary buttons are for alternative actions
  • Tertiary buttons for supplementary actions
  • Maintain consistent spacing between buttons (16px)
  • Use appropriate size based on context and importance

Accessibility

  • All buttons have visible focus states
  • Color contrast meets WCAG 2.1 AA standards
  • Interactive states are clearly visible
  • Disabled states maintain sufficient contrast
