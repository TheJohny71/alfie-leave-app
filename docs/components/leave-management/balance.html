Balance Display

Overview

The Balance Display component provides a visual representation of leave balances with progress tracking and expiry alerts.

Usage

import { LeaveBalance } from './components/LeaveManagement';

const balances = [
    {
        type: 'Annual Leave',
        used: 10,
        total: 20,
        pending: 2,
        expiryDays: 90
    }
];

Props

Prop Type Required Description
balances array Yes Array of balance objects
