Approval Interface

Overview

The Approval Interface provides managers with a comprehensive view of leave requests and team coverage.

Usage

import { LeaveApproval } from './components/LeaveManagement';

const request = {
    employee: 'John Doe',
    department: 'Engineering',
    startDate: '2024-12-20',
    endDate: '2024-12-25',
    duration: 5,
    reason: 'Annual vacation',
    teamCoverage: [
        { name: 'Jane Smith', available: true },
        { name: 'Mike Johnson', available: false }
    ]
};

 handleApprove(comments)}
    onReject={(comments) => handleReject(comments)}
/>

Props

Prop Type Required Description
request object Yes Leave request details
onApprove function Yes Callback for approval action
onReject function Yes Callback for rejection action

Features

  • • Team coverage visualization
  • • Comments and feedback system
  • • Confirmation dialogs for actions
